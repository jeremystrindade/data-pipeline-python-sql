# Data Pipeline Project (Python + SQL + Airflow)

## Overview
This project focuses on building an end-to-end data pipeline, covering the full data lifecycle:
- Data ingestion from external APIs
- Data transformation using Python
- Storage in a relational database
- Pipeline orchestration using Apache Airflow

## Tech Stack
- Python
- SQL (PostgreSQL)
- Apache Airflow

## Architecture (Planned)
External API  
↓  
Python ingestion script  
↓  
PostgreSQL database  
↓  
Airflow DAG orchestration
